## First Steps

Start by reading the [Documentation]{:.heading.flip-title}.

containing example content:

```
├── licenses
├── assets

│       ├── docs
│       └── projects
├── CHANGELOG.md
├── forms-by-example.md
├── LICENSE.md
└── NOTICE.md
```
